О проекте

Данный проект представляет собой интерактивную галерею автомобилей с применением современных веб-технологий: HTML, CSS и JavaScript. Основная цель работы — продемонстрировать реализацию фильтрации изображений, анимаций и динамического взаимодействия с элементами интерфейса на стороне клиента.
Сайт выполнен в виде трёх логически связанных страниц:
1 - Главная страница — содержит общее описание проекта и навигационное меню.
2 - Галерея — основной раздел, где представлены изображения автомобилей различных категорий (спорткары, седаны, внедорожники). Галерея поддерживает фильтрацию по категориям.
3 - О проекте — содержит цель работы. Показывает, какие эффекты можно применить, как облегкчить работу и функционал для проекта.


Преимущества проекта:
Минимальная нагрузка на систему (работает без серверной части).
Современный дизайн и удобный интерфейс.
Лёгкость масштабирования — можно легко добавить новые категории и изображения.

Контрольные пункты

  1. HTML: header, main, section, article, footer
  2. CSS: Grid / Flex, адаптивность, transition/transform
  3. JS: события, массив данных, DOM-manipulation, localStorage
